# 📚 Sistema Web Educativo

Un sistema web desarrollado con **Java Spring Boot** para mejorar el aprendizaje de los alumnos mediante la gestión de cursos, usuarios y contenidos educativos.  
Este proyecto busca ofrecer una plataforma moderna, escalable y fácil de usar para instituciones educativas.

---

## 🚀 Características principales
- Gestión de usuarios (alumnos, docentes, administradores).
- Registro y autenticación segura con Spring Security.
- Administración de cursos, módulos y contenidos.
- Integración con base de datos relacional (MySQL/PostgreSQL).
- Arquitectura basada en **MVC** con Thymeleaf para las vistas.
- API REST para futuras integraciones.

---

## 🛠️ Tecnologías utilizadas
- **Java 21**
- **Spring Boot** (Web, Data JPA, Security)
- **Thymeleaf** (motor de plantillas)
- **MySQL / PostgreSQL** (base de datos)
- **Maven** (gestión de dependencias)
- **GitHub** (control de versiones y colaboración)
